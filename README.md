#  Прогноз спроса на сельскохозяйственную продукцию с использованием временных рядов

Проект посвящён прогнозированию объёмов сельскохозяйственного производства и спроса на основе открытых рыночных данных с применением классических статистических моделей и нейросетевых методов анализа временных рядов.

---

##  Цель проекта

Разработать систему прогнозирования для сельхозпродукции (пшеница, яблоки, молоко и др.), которая:

- анализирует временные ряды производства и спроса  
- учитывает долгосрочные тренды и сезонность  
- сравнивает базовые статистические методы с нейросетями  
- оценивает качество прогноза по метрикам MAPE и SMAPE  
- формирует прогноз на будущие периоды  

---

##  Источники данных

### FAOSTAT (Продовольственная и сельскохозяйственная организация ООН)

Открытая база статистики сельского хозяйства:

https://www.fao.org/faostat/

Используемые поля:
- Year — год  
- Area — страна  
- Item — продукт  
- Value — объём производства  

*(В дальнейшем возможно подключение данных USDA Market News для более высокой частоты наблюдений.)*

---

## Реализованные модели

| Модель | Назначение |
|-------|-----------|
| ARIMA | Базовая статистическая модель временных рядов |
| Prophet | Модель трендов и сезонности |
| N-BEATS | Нейросетевая модель прогнозирования временных рядов |
